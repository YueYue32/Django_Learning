# 建立應用程式 application (app)

application 可視作為本專案建立的套件，通常會有多個 app 進行搭配，負責不同的功能。

語法：

進入專案目錄 myweb

    cd myweb

建立第一個 Django app : myapp

    py manage.py startapp <app名稱>
